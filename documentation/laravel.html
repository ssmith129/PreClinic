Installation Guide

After unzipping Template Files, you will find our product Preclinic->Laravel(Source code), Documentation in the following Main Files folder.

Once unzipped the files, run the command to install the composer packages: 

composer install (or) composer update

In the root directory, you will find a file named .env.example, copy the content,create a new file and paste that content save as .env and run the following command to generate the key. 

php artisan key:generate
php artisan migrate

Open the terminal in your root directory(preclinic) & and run the following command for install all dependencies and build the project:

npm install
npm run build

For running a project, run the command: 

php artisan serve
